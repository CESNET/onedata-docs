title: "Manage Files via the Onedata Web Portal"
---

This guide shows you how to use the Onedata Web Portal to upload, browse, share, and manage files directly in your browser. For a complete feature reference, see the [official Onedata documentation](https://onedata.org/#/home/documentation/25/user-guide/interfaces/web-file-browser.html).

## Prerequisites

- An active e-INFRA CZ account with access to at least one Onedata Space.
- A modern web browser with pop-ups enabled.
- The Onedata portal URL: [https://onedata.e-infra.cz](https://onedata.e-infra.cz)

> **Tip:** Pop-ups must be allowed for file downloads and some upload workflows to function correctly.

---

## Procedure

### 1. Log in

Open [https://onedata.e-infra.cz](https://onedata.e-infra.cz) and log in via one of the available identity providers.

### 2. Browse your Spaces

After logging in, you will see your Spaces listed.

- Click a **Space** to open it.
- Navigate folders like a standard file manager.
- Click folders to enter them.

### 3. Work with files

#### Upload files

1. Open the target directory inside your Space.
2. **Drag and drop** files into the browser window, or use the upload button.

#### Download files

- **Double-click** a file to open or download it.
- Alternatively, use the download option in the file context menu.

#### Create folders

- Use the **New folder** button in the toolbar.

#### Rename or delete

- Right-click a file or folder and select **Rename** or **Delete** from the context menu.

### 4. Attach metadata

You can add metadata to help organize and describe your data:

- To a **Space** — select the Space and open the metadata panel.
- To a **Folder** — right-click the folder and choose the metadata option.
- To a **File** — right-click the file and choose the metadata option.

### 5. Manage sharing and access

#### Control who can access your data

- Permissions are managed per Space or per file.
- Add users or groups and assign read/write access as needed.

#### Share files publicly

- Select the file or folder you want to share.
- Configure public access through the sharing settings.
- Remember: access level depends on the permissions you assign.

## When to use the Web Portal

| Use the Web Portal when... | Consider an alternative when... |
|---|---|
| You need to quickly upload or download a few files | You need to transfer large datasets (use [Oneclient](/docs/how-to/how-to-use-client)) |
| You want to browse data without installing anything | You need automation or scripting (use APIs) |
| You are performing basic file operations | You need programmatic access (use REST or S3 API) |

> **Tip:** For day-to-day file management and collaboration, the Web Portal is the fastest way to get started. For HPC workflows or large-scale data movement, switch to [Oneclient](/docs/how-to/how-to-use-client) or the API interfaces.
